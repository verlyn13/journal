    (static) createEditor(parentElement, initialDoc, options) → {EditorView}

    Creates a CodeMirror editor instance in the given parent element. This is the main function for initializing the Markdown editor. It configures the editor with a custom set of extensions providing core functionality (line numbers, history, keymaps), Markdown language support, the application's custom theme, placeholder text, and a callback mechanism for content changes.
