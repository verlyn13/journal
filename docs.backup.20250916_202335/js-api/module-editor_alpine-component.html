    (inner) init()

    Initializes the editor component. Sets up the editor state, loads initial content, and establishes auto-save functionality. Uses a MutationObserver to reliably find the target DOM element (`x-ref="editorElement"`) before creating the CodeMirror instance, ensuring initialization occurs correctly even with potential DOM rendering delays.
