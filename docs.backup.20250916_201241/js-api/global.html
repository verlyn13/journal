    insertMarkdownSyntax(view, prefix, suffixopt, placeholderopt, blockopt)

    Original generic function for inserting Markdown syntax, handling both inline and block elements. Kept primarily for compatibility with older/complex buttons (Image, Table, Code Block). Handles wrapping selected text or inserting placeholders with prefixes/suffixes. For block elements, ensures prefixes/suffixes are placed on appropriate new lines.
    Source:
    • editor/toolbar-actions.js, line 134
    Parameters:
    Name Type Attributes Default Description
    view EditorView The CodeMirror EditorView instance.
    prefix string The syntax to insert before the selection or placeholder (e.g., '```\n').
    suffix string <optional>
    		 '' The syntax to insert after the selection or placeholder (e.g., '\n```').
    placeholder string <optional>
    		 '' Text to insert between prefix and suffix if there's no selection.
    block boolean <optional>
    		 false If true, treats the insertion as a block element, managing newlines around the prefix and suffix.
